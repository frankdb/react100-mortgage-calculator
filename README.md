# React 100 Mortgage Calculator

This project is a simple mortgage calculator built with React and styled with Bootstrap.

## Getting Started

- "npm run build"

- "npm start"

- Open on localhost:3000

### Installing

Run "npm install" in Terminal

## Running the tests

Run "npm test"

## Built With

- React

- Bootstrap
